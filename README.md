# DevOps Journey

A modern, responsive developer portfolio and learning tracker built with HTML, CSS, and Vanilla JavaScript.

## Overview

DevOps Journey is a personal website designed to document and showcase my learning progress as I work toward becoming a Cloud & DevOps Engineer.

The website tracks:

* Learning roadmap progress
* Technical skills
* Learning milestones
* DevOps projects
* Growth statistics

## Features

### Hero Section

* Personal introduction
* Current learning status
* Career goal display

### Learning Roadmap

Track progress in:

* Git & GitHub
* Linux
* Docker
* Kubernetes
* AWS
* CI/CD

### Skills Dashboard

Display current technical skills with animated cards.

### Learning Timeline

Show completed milestones and upcoming goals.

### Projects Portfolio

Store and showcase DevOps projects with:

* Project name
* Description
* Tech stack
* GitHub repository links

### Statistics Dashboard

Track:

* Repositories created
* Projects built
* Learning days
* Technologies learned

### Responsive Design

Works perfectly on:

* Desktop
* Tablet
* Mobile

## Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
